# AWS S3 Terraform Configuration

This Terraform project manages AWS S3 resources including bucket creation and object uploads.

## Prerequisites

- [Terraform](https://www.terraform.io/downloads) (version 1.0+)
- [AWS CLI](https://aws.amazon.com/cli/) installed and configured
- AWS account with appropriate permissions for S3

## Resources

This configuration creates:

- **S3 Bucket** (`aws_s3_bucket.default`) - An S3 bucket
- **S3 Object** (`aws_s3_object.object`) - Uploads `my_file.txt` to the bucket

## Setup

1. **Configure AWS credentials:**
   ```bash
   aws configure
   ```

2. **Initialize Terraform:**
   ```bash
   terraform init
   ```

3. **Review the plan:**
   ```bash
   terraform plan
   ```

4. **Apply the configuration:**
   ```bash
   terraform apply
   ```

## Usage

### Apply changes
```bash
terraform apply --auto-approve
```

### Destroy resources
```bash
terraform destroy
```

### View current state
```bash
terraform show
```

## Files

- `main.tf` - Main Terraform configuration
- `.gitignore` - Git ignore rules for Terraform files
- `my_file.txt` - File to be uploaded to S3

## Notes

- Ensure `my_file.txt` exists in the project directory before applying
- The S3 bucket name is auto-generated by AWS
- All state files are excluded from version control via `.gitignore`
